## Kompilator

Kompilator dla prostego jeyzka imperatywnego do kodu maszyny wirtualnej.

## Wymagania

Biblioteka sly

## Użycie

`python kompilator.py <in> <out>`

gdzie:

-`[in]`-scieżka do kodu w kompilowanym jezyku imperatywnym
-`[out]`-scieżka do pliku gdzie ma zostać zapisany zkompilowany kod maszyny wirtualnej
